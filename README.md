# Top Bar

`topbar.koplugin` adds customizable status modules to the File Browser top bar.

## Features

- Modular top bar content with custom order.
- Built-in modules:
  - Wi-Fi
  - SSH
  - Memory
  - Storage (free space for current path)
  - Custom text
  - Clock
  - Battery
  - Brightness
  - Warmth
  - Up time
  - Awake time
  - Suspend time
- Display mode:
  - `Active only`
  - `Always`
- Optional sync to TouchMenu bottom info text (drop-down menu).

## Where To Configure

Open File Browser menu and go to:

`Settings -> Top Bar Settings`

Inside this menu:

- `Modules`: enable/disable modules, sort module order.
- `Configuration`: display mode, separator, clock refresh, path visibility, bold text, and drop-down sync.

## Notes

- Works normally on Kindle Paperwhite 11th gen (PW5).
